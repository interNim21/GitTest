글자관련태그

b 태그 : 글자의 굵기를 표기하는 태그
i 태그 : 글자를 기울이는 태그 HELLO MSTER MY YESTERDAY


strong 태그 : 중요성,강조성 태그 em 태그 : 문장 일부분의 중요성을 강조할 때 사용 됌

mark 태그 : 형광펜 효과를 나타냄
u 태그 : under line (밑줄)을 나타냄
s 태그 : 취소선을 나타냄(잘못됀 걸 강조)
del 태그 : 삭제선(없어진 텍스트)
small 태그 : 작은 글자 표시
sup 태그(윗첨자) : x2
sub 태그 (아랫첨자) : log10
abbr 태그 : 마우스 오버 시 툴팁 출력
HTTP

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수 도 있고, 기울이거나 취소선을 넣는 등
다양하게 사용할 수 있다.
