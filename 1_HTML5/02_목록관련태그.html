목록 관련 태그

ul(Unordered List) 태그 : 순서가 없는 목록
ol(Ordered List) 태그 : 순가가 있는 목록
li(List item) 태그 : ul, ol 태그 내부에 들어가 목록의 요소를 나타내는 태그

ol 태그

ol 태그의 type 속성
type="a" -> 소문자 알파벳
type="A" -> 대문자 알파벳
type="i" -> 소문자 로마 숫자
type="I" -> 대문자 로마 숫자
type="1" -> 숫자(기본값)
start 속성 : 시작 번호 지정
reversed 속성 : 시작 순서 반대로 시작하는 속성
